## Overview

This repository contains the complete R analysis pipeline used in the manuscript. The study performs a systematic in silico comparative analysis of circular RNA (circRNA) expression across three breast cancer subtypes:

| Subtype | GEO Accession | Samples |
|---|---|---|
| Triple-Negative Breast Cancer (TNBC) | [GSE142731](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE142731) | 42 |
| HER2-Positive (HER2+) | [GSE191230](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi=GSE191230) | 20 |
| Inflammatory Breast Cancer (IBC) | [GSE207248](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi=GSE207248) | 44 |

**Total: 106 RNA-seq samples**

The pipeline covers:
- circRNA abundance quantification and normalization (SRPBM)
- Quality control (depth, burden, PCA/MDS)
- BED file generation and database annotation (circBase, CircAtlas, MiOncoCirc)
- Novel circRNA identification
- circRNA–miRNA–mRNA interaction network construction
- Differential expression analysis (DESeq2)
- Functional and pathway enrichment visualization (DAVID/GO/KEGG)


## Study Design

Raw FASTQ files were quality-controlled with **FastQC** and **123FASTQ**, aligned to **GRCh38** using **TopHat2**, and circRNAs were identified with **CIRCexplorer2** and quantified with **CIRIquant**. The R pipeline in this repository takes the CIRIquant output (per-sample expressionsfinal.txt files) and performs all downstream analyses described in the manuscript.

Raw FASTQ (GEO)
    ↓  FastQC / 123FASTQ (QC & trimming)
    ↓  TopHat2 (alignment → GRCh38)
    ↓  CIRCexplorer2 (circRNA detection)
    ↓  CIRIquant (quantification → expressionsfinal.txt)
    ↓
    ← THIS REPOSITORY STARTS HERE →
    ↓
Stage 1:  Read filtering (≥2 reads in ≥1 sample)
Stage 2:  SRPBM normalization
Stage 3:  Quality control (PCA, MDS, depth, burden)
Stage 4:  BED generation (hg38)
Stage 5:  Database overlap (circBase / CircAtlas / MiOncoCirc)
Stage 6:  Read support distribution + donut chart
Stage 7:  Exon structure + chi-square test
Stage 8:  Chromosomal SRPBM density
Stage 9:  Venn diagram (shared high-confidence circRNAs)
Stage 10: SRPBM/FPKM integration + heatmap + PCC
Stage 11: circRNA–miRNA–mRNA network + Cytoscape export
Stage 12: GO/KEGG enrichment plotting (DAVID outputs)

**External tools (not R — required for upstream steps only):

| Tool | Version | Purpose |
|---|---|---|
| FastQC | ≥0.11 | Raw read QC |
| 123FASTQ | — | trimming |
| TopHat2 | 2.1.x | Read alignment (GRCh38) |
| CIRCexplorer2 | ≥2.3 | circRNA detection |

## Data Access

### RNA-seq raw data (GEO)
The three datasets used in this study are publicly available at NCBI GEO:

| Dataset | URL |
|---|---|
| GSE142731 (TNBC) | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE142731 |
| GSE191230 (HER2+) | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi=GSE191230 |
| GSE207248 (IBC) | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi=GSE207248 |

Download raw FASTQ files from SRA, align with TopHat2, and run CIRIquant to generate the expressionsfinal.txt files required by this pipeline. Alternatively, processed count files (CIRIquant output) are available upon request from the corresponding author.

### Reference databases
Download reference circRNA databases and place them in data/reference/:

| Database | Source | File |
|---|---|---|
| circBase (hg19 → hg38 lifted) | http://www.circbase.org/ | `circBase_Homo_sapiens_hg38_lifted.bed` |
| CircAtlas v3.0 | https://circatlas.biols.ac.cn/ | `CircAtlas_human_bed_v3.0.zip` |
| MiOncoCirc | https://mioncocirc.github.io/ | `MiOncoCirc_all_hg38.bed` |

> CircBase coordinates were originally in hg19 and were lifted to hg38 using the UCSC `hg19ToHg38` chain file via the `rtracklayer::liftOver()` function (handled automatically in Stage 5).

### miRNA interaction data
- **miRBase** (high-confidence mature miRNAs): https://www.mirbase.org/
- **miRTarBase** (experimentally validated miRNA–mRNA interactions): https://mirtarbase.cuhk.edu.cn/

### DAVID enrichment
Functional enrichment analyses (GO and KEGG) were performed using the DAVID web tool (https://david.ncifcrf.gov/, v6.8). Export DAVID chart results as `.xlsx` files and place them in the `data/processed/stage07/` directory before running Stage 12.

## How to Run

### Full pipeline (Stages 1–12)

After preparing all input files (see [Data Access](#data-access)), open R or RStudio in the repository root and run:

r
source("circRNA_network_analysis.R")


The script uses the `here` package so it automatically detects the repository root. All outputs are written to `data/processed/` and `outputs/`.

### Running individual stages

Each stage is clearly delimited by comments in the script. You can run them sequentially by sourcing the file, or copy-paste individual stage blocks into an interactive R session. Stages must be run **in order** (each stage depends on outputs from previous stages).

## Pipeline Stages

| Stage | Description | Key Outputs |
|---|---|---|
| **1** | Filter circRNA counts (≥2 reads in ≥1 sample) | `stage01/<GSE>_filtered_counts_ge2.csv` |
| **2** | SRPBM normalization + additional filters (mean SRPBM ≥1, present in ≥2 samples) | `stage02/<GSE>_filtered_counts_ge2_SRPBM_ge1.csv` |
| **3** | Quality control: sequencing depth, circRNA burden, PCA, MDS | `outputs/figures/QC_*.pdf` |
| **4** | BED file generation (hg38 coordinates) | `stage04/<GSE>_final_hg38.bed` |
| **5** | Database overlap vs. circBase, CircAtlas, MiOncoCirc; novel circRNA identification | `stage05_overlap_summary_*.csv`, `NOVEL_circRNAs_*.csv` |
| **6** | Read support distribution histogram + donut chart | `Fig3a_*.png`, `Fig3b_*.png` |
| **7** | Exon structure analysis + chi-square test | `stage07_*.csv` |
| **8** | Chromosomal SRPBM density | `stage08_*.pdf` |
| **9** | Venn diagram of shared high-confidence circRNAs across subtypes | `Venn_*.pdf` |
| **10** | SRPBM/FPKM integration, Pearson correlation, heatmap | `stage10_*.pdf`, `stage10_PCC_*.csv` |
| **11** | circRNA–miRNA–mRNA network construction + Cytoscape export (node/edge tables) | `Network_triplets_*.tsv`, Cytoscape-compatible files |
| **12** | DAVID GO/KEGG enrichment bar plots + circRNA×KEGG heatmap | `Fig_BP/MF/CC/KEGG.pdf`, `Fig_circRNA_KEGG_heatmap.pdf` |
